about()

Hi, my name is Jesse. I'm a man of simple tastes. I like computers, nature, and travelling. Lately I've been spending most of my time programming, surfing or snowboarding.

I'm a minimalist at heart. A philosophy I believe in is software should be made as simple as possible, but no simpler. Writing simple software that 'Just Works'™ is indeed difficult but, as engineers, we must collectively rise to the challenge.

interests()

I've always had an itch to build developer tooling, and I've developed a strong interest in compilers recently. I've been working on a metaprogramming system for the C programming language. It's called poof, and can be found in the open_source section

In the past I've been inspired by the work of Casey Muratori on Handmade Hero and decided to write a 3D game engine from scratch. It's called Bonsai, and can can be found in the open_source section.

Outside of writing software I can be found backcountry skiing, snowmobiling, surfing, wrenching on cars, and driving them!

open_source()

poof - A practical metaprogramming environment for the C programming language.

Bonsai - A 3D game engine focused on fast-paced, arcade style games. Written in C with no dependencies.
